# VSCode

## 개요

- 비주얼 스튜디오 코드는 마이크로소프트가 개발한 범용 코드 편집기이다.

## 단축키

- 프로젝트 익스플로러 열기 `Shift+Ctrl+E`
- 폰트 축소 확대 `CTRL+-` `CTRL++`

## 내가 지정한 단축키

- 커밋 `Ctrl+K`
- 대소문자 전환 `Shift+Ctrl+C` (String Manipulation 확장)
- md 파일에서 클립보드 이미지 붙여넣기 `Ctrl+Alt+V` (Markdown Paste 확장)
