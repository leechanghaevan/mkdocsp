# Three.js

> Three.js는 사유 브라우저 플러그인에 의존하지 않고 자바스크립트 언어를 사용하여 웹사이트의 한 부분으로서 그래픽 처리 장치(GPU) 가속 3차원 애니메이션을 만들 수 있게 한다.[3][4] 이는 WebGL의 출현으로 인하여 가능하게 되었다. - 위키백과
