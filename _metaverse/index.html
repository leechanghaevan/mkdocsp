1.3.1. 타입 스크립트 설정

tsconfig.json

"target": "es5"
"sourceMap": true
"outDir": "./dist",
"rootDir": "./_src",
  • import 문 변환하지 않고 남겨둠.
  • 단점: 임포트 경로가 지저분하다. import * as THREE from '../../../node_modules/three/src/three.js
  • 장점: commonjs 없이 동작한다.
"module": "es2015"
  • import 문을 commonjs 형태로 변환한다.
  • 단점: commonjs 모듈을 번들러를 통해 묶어 주어야 한다.
  • 장점: 임포트 경로가 깔끔하다. import * as THREE from 'three
"module": "commonjs"
"moduleResolution": "node"

타입스크립트 컴파일시 팝업창 안뜨고 조용히 진행되게
tasks.json

"presentation": {
"reveal": "silent",
"clear": true,
"showReuseMessage": false
}
