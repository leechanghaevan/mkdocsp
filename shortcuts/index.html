1. 단축키 모음

1.1. 공통 단축키

  • 전체창 f11

1.2. 웹 브라우저

  • 주소창 바로가기 ctrl + L
  • 주소창 바로가기 alt + D
  • 이전페이지 alt + left
  • 다음페이지 alt + right
  • 개발툴 f12

1.3. Vi 에디터

  • 편집모드 전환 i
  • 편집모드 종료 ESC
  • 종료 :q
  • 저장 후 종료 :wq
  • 저장하지 않고 종료 :q!

1.5. VSCode (사용자 정의)

  • 커밋 Ctrl + K
  • 대소문자 전환 Shift + Ctrl + C (String Manipulation 확장)
  • md 파일에서 클립보드 이미지 붙여넣기 Ctrl + Alt + V (Markdown Paste 확장)

1.6. Windows 단축키

  • 이모지😊 입력 Window + ;
  • 파워셀 보안 정책 축소 set-executionpolicy unrestricted

1.7. 크롬

  • 개발툴 켜기 f12
